This is a Library Management Web Application using flask. 
The first step is to install all the necessary packages. All the packages will be mention in the requirement section. 
Make sure your pip version is up to date because as most of all the packages require latest pip version to install the packages. 
Whenever trying to run this program make sure that you env (environment) is always on. 
To turn your environment on type this command  => .\env\Scripts\activate.ps1
Once your environment is on you can run the code using this command  => python .\main.py 

Once the code starts to run, you will recieve a address. 
You can copy paste that address on to google or can press (ctrl + click) to open that address which will redirect you to the website.
